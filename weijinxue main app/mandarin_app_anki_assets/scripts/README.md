# APKG Audio Assets

This folder was generated from your Anki `.apkg` deck.

Copy these into your Vite React app:

- `public/anki-audio/` → `your-app/public/anki-audio/`
- `src/data/audioMap.json` → `your-app/src/data/audioMap.json`
- `src/data/hskDeckEntries.json` → `your-app/src/data/hskDeckEntries.json`
- `src/components/AudioButton.jsx` → `your-app/src/components/AudioButton.jsx`

Then import `AudioButton` wherever you show a selected character or word.
